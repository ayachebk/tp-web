Virtualization

In computing, Virtualization refers to the act of creating a virtual (rather than actual) version of something.
This include virtual computer hardware, virtual storage devices and virtual network resources.

Levels of Virtualization

  1. Virtualization at Instruction Set Architecture (ISA) level

    2. Every machine has an instruction set. This instruction set is an interface between software and hardware. Using this instructions software can
    communicate with hardware.

  2. Virtualizaion at Operating System(O.S.) level

    3. In Virtualizaion at HAL level each virtual machine is built from scratch i.e. by installing O.S., application suites, networikng systems, etc.

    • In cloud sometimes, we need to initialize 100 Virtual machines at a single time, if we use virtualization at Hardware abstraction layer
      (HAL) level this can take to much time.
    • So, to overcome this in Virtualizaion at Operating systems level we share operation system between Virtual machines along with the
      hardware
